Inicio

Zona Cardio

Este es un espacio el cual cuenta con 5 maquinas caminadoras, 5 elipticas, varis bicicletas e incluso un espacio en el cual se puede saltar el laso con el fin de ganar una mejor resistencia cardio bascular, y ayudar a quemar calorias dependiendo de lo que se quiera hacer ya sea definir tu cuerpo o perder peso.

Zona de Musculacion

En esta zona cuentas con maquinas tales como una maquina la cual la puede usar para hacer press de banca o sentadilla con peso varias mancuernas con diferentes tipos de pesos que van de 5 lbs. hasta las 60 lbs para poder hacer el ejercicio que quieras con total libertad una maquina polivalente para cambiar de ejercicios y de grupos musculares, cuenta tambien con 4 bancas para el uso que quiras
